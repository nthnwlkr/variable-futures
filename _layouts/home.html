Variable Futures: A Typographic Exploration
is an experiment with variable fonts in both web and print contexts.

This website is best experienced alongside the printed publication. If you don't have one lying around, scroll down to see a few images. Variable Futures: A Typographic Exploration aims to integrate variable fonts into the context of the web and related technologies.

Variable Futures is presented in three distinct exploration spaces; firstly, we display our findings from a solely conceptual basis, building on previous work from significant contributors in the field of variable fonts.

Secondly, we experiment in two- and three-dimensional capacities, expanding the Variable Futures exploration frontier.

Finally, we bind together all previous findings to produce a series of experimental variable font tools primarily for use on the web.
